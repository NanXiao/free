# free
[free](https://linux.die.net/man/1/free) command for `OpenBSD`:

	# ./free
	                     total              used              free
	Mem:                 4032M               28M             4004M
	Swap:                   0B                0B                0B

## Usage  

	# git clone https://github.com/NanXiao/free.git
	# cd free
	# make
	# ./free
